# Bones::RPC::Adapter::Erlang

Bones::RPC erlang adapter for ruby

## Installation

Add this line to your application's Gemfile:

    gem 'bones-rpc-adapter-erlang'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install bones-rpc-adapter-erlang
